# todolist_app
Simple Flask Todo App using SQLAlchemy and SQLite database.

Install Flask
```
$ pip install Flask
$ pip install Flask-SQLAlchemy
```
run 
```
python3 app.py
```
then you can see Todo App at http://127.0.0.1:5000
